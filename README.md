# baseball-scoreboard

A console application aimed to diaplay the baseball scoreboard

## Function

- Show the score board
- Control
    - pitcher's pitch result
        - strike(`s`), ball(`b`), pickoff(`pk`)
    - hitter's hit result
        - `1b`, `2b`, `3b`, `hr`
    - runner's steal base
        - `s1b`, `s2b`, `s3b`
    - filder's error
        - `e`
        - assume all runner get into next base
